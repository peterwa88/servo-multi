# DECISIONS

## D-001
Use official Servo source in `servo_origin/` instead of rebuilding a browser engine from scratch.

## D-002
Keep all new shell/testing/orchestration code in `servo_src/`.

## D-003
Treat upstream Servo updates as a controlled sync operation, not ad hoc copy-paste.

## D-004
Prefer shell-layer/wrapper-layer development before engine-internal modifications.
